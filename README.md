# 🚀 Getting Started

Clone this repository

```bash
git clone https://github.com/CoderXash9/Django-Learning.git
```

Move into the project

```bash
cd Django-Learning
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

Install Django

```bash
pip install django
```

Run Server

```bash
python manage.py runserver
```
